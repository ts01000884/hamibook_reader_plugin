## 詳細說明（Detailed description，貼到「產品詳細資料」欄位）

```
HamiBook 悅讀助手，讓你在 HamiBook WebReader 上「用聽的」讀書，並提供更護眼的夜間閱讀體驗。

■ 主要功能
・中文語音朗讀：朗讀目前頁面內容，語速可調（上限 1.5 倍），適合通勤、做家事或讓眼睛休息時聆聽。
・播完自動翻頁接續：一段念完會自動翻到下一頁繼續朗讀，中途純圖片頁也能順利接續，不用一直手動翻頁。
・閱讀進度記憶：記住整章段落與上次念到的位置，下次打開接著聽。
・舒適夜間模式：一鍵切換介面深色主題，降低夜間閱讀的亮度負擔；切換章節時畫面穩定不再閃黑。
・平滑翻頁：可從瀏覽器工具列 ICON 手動開啟，預先修正前後頁並抑制原生遮罩外露，減少固定版面 EPUB 翻頁時的白畫面跳動；首次預設關閉。
・浮動控制面板：右下角小面板可播放/暫停、調整語速、開關夜間模式，可縮放、可收合，不擋閱讀。

■ 適合誰
・想用聽的方式讀電子書的人。
・長時間閱讀、需要護眼夜間模式的人。
・希望朗讀能自動翻頁、不用一直操作的人。

■ 使用方式
安裝後，開啟 HamiBook WebReader 的閱讀頁即可看到右下角控制面板（語音朗讀面板僅在 EPUB 文字格式顯示）。點播放開始朗讀，點「日／夜」按鈕切換夜間模式。固定版面 EPUB 可點擊瀏覽器工具列的擴充 ICON，再開啟「平滑翻頁」。

■ 生效範圍
本擴充僅在 HamiBook WebReader 閱讀頁（webreader.hamibook.com.tw/viewer/）運作，其他網站完全不會載入。

■ 隱私
本擴充不要求額外 Chrome API 權限。為提供朗讀、夜間模式與平滑翻頁，擴充會在你的瀏覽器本機處理 HamiBook 閱讀頁內容與目前閱讀頁位置；這些資料不會傳送給開發者、分析服務、廣告服務或其他第三方。所有設定（閱讀進度、夜間模式與平滑翻頁開關）僅儲存在你自己的瀏覽器中；平滑翻頁啟用時只會向 HamiBook 本身預載前後閱讀頁。診斷 LOG 只在使用者主動按下複製時交給剪貼簿，不會自動上傳。

■ 說明
本擴充為第三方輔助工具，與 HamiBook／中華電信無任何隸屬或合作關係，僅在你已具備閱讀權限的頁面上，提供朗讀與顯示上的輔助。語音由瀏覽器內建的語音合成功能提供，實際音色與可用語音依作業系統而定。
```

## 隱私權實務（Privacy practices，審查會問）
- 單一用途說明：在 HamiBook WebReader 閱讀頁提供語音朗讀、夜間閱讀與 EPUB 平滑翻頁等輔助功能。
- 資料處理揭露：會在本機處理 **網站內容**（HamiBook 閱讀頁文字與版面資源）及 **瀏覽活動**（目前 HamiBook 閱讀頁位置），僅用於朗讀、進度、夜間模式與平滑翻頁；不會傳送給開發者或第三方。
- 權限說明：未宣告 `permissions` 或 `host_permissions`；僅以 content script 的 `matches` 限定 HamiBook 閱讀頁，並讀寫同源 localStorage 保存偏好與進度。
- 遠端程式碼：**未使用**（所有程式碼隨擴充封裝，無遠端載入）。

## 網站存取權限請求說明（Host permission justification）

```
本擴充為單一用途的閱讀輔助工具，僅透過 content_scripts 的比對模式在 HamiBook WebReader 的閱讀頁（https://webreader.hamibook.com.tw/viewer/*）注入，不使用任何 permissions，也未要求任何額外 host_permissions，不會在其他網站執行。

需要此網站存取範圍，是因為擴充的核心功能都必須直接在該閱讀頁上運作：
1. 語音朗讀：讀取頁面上的內文段落，交給瀏覽器內建語音合成功能朗讀。
2. 自動翻頁接續：偵測目前段落是否已顯示，並點擊頁面上的「下一頁」按鈕讓朗讀連續進行。
3. 夜間模式：注入 CSS 並調整頁面樣式，提供護眼的深色閱讀介面。
4. 閱讀進度記憶：以同源 localStorage 記住整章段落與上次朗讀位置。
5. EPUB 平滑翻頁：僅在固定版面閱讀頁（/viewer/07/）由使用者透過擴充工具列 ICON 手動開啟；依目前視窗尺寸預先修正相鄰閱讀頁，並在原生 EPUB FIX 完成前進行視覺交接，以減少白畫面與版面跳動。此功能首次預設關閉。
6. 本機診斷 LOG：在記憶體中保留最多 200 筆翻頁狀態、耗時與版面尺寸，只有使用者主動點擊複製時才會放入剪貼簿，不會自動上傳；LOG 不包含書名、URL 查詢參數、會員資料、token 或書籍內文。

以上功能僅在使用者已具備閱讀權限的 HamiBook 閱讀頁上進行顯示、朗讀與翻頁輔助。閱讀頁內容與目前閱讀位置只在使用者裝置本機處理，不會傳送給開發者、分析服務、廣告服務或其他第三方。平滑翻頁啟用時，只會向目前已登入的 HamiBook WebReader 同源站台預載相鄰閱讀頁，不連線任何第三方伺服器。

存取範圍已限縮到 webreader.hamibook.com.tw 單一網域的 /viewer/ 路徑；唯一的路徑萬用字元只用於涵蓋該路徑下的閱讀頁，未使用 <all_urls>、其他網域或更廣泛的主機比對模式。
```
