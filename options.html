TTS 伺服器設定

本機 TTS 只需填 Port；外部 TTS 才需要完整的伺服器資料。

伺服器類型

朗讀文字傳送說明

使用伺服器 TTS 時，目前朗讀的文字會傳送到這台電腦的本機 TTS Server，以產生語音。

  • 使用瀏覽器內建語音時，不會向 TTS Server 傳送內文。
  • 本擴充沒有開發者營運的 TTS、分析或廣告後端；開發者不會收到朗讀文字。
  • 外部服務可能自行保存請求；使用 HTTP 時，文字與 API Key 不具傳輸加密。

使用提示

  • 本機模式會自動使用 http://localhost:8890/v1。
  • API Key 留空、模型固定使用 kokoro，不需要手動設定。
  • 連接其他電腦、ZeroTier 或雲端服務時，請切換到「外部 TTS」。
